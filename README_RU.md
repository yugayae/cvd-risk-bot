## 🎯 Обзор

Интеллектуальный Telegram-бот для оценки риска сердечно-сосудистых заболеваний на основе **машинного обучения CatBoost** с **интерпретируемостью SHAP**. Поддержка языков: **русский, английский, корейский**.

### Ключевые особенности

- 🤖 Интерактивный Telegram-бот
- 🧠 Машинное обучение (CatBoost, ROC-AUC 80%)
- 📊 Объяснения SHAP
- 🌍 3 языка (ru/en/kr)
- ⚕️ Клинический уровень точности
- 🔒 Конфиденциальность с согласием пользователя
- 📈 Автоматический расчет ИМТ
- 📊 Логирование в Google Таблицы

## 🔒 Конфиденциальность и Сбор Данных

### Система Согласия

- ✅ **Прозрачное согласие** перед оценкой
- ✅ **Только анонимные данные** (без Telegram ID, имени, контактов)
- ✅ **Добровольное участие** - отказ не влияет на результат
- ✅ **Научная цель** - улучшение точности модели

### Что Собирается (при согласии)

- Медицинские данные (возраст, давление, холестерин и т.д.)
- Результаты оценки и значения SHAP
- Регион проживания (например, "Восточная Азия")
- Временная метка

### Что НЕ Собирается

- ❌ Telegram username/ID
- ❌ Телефон/email
- ❌ Личные идентификаторы
- ❌ История переписки

### Хранение Данных

- Данные записываются в **Google Таблицы** через webhook
- **Соответствие GDPR** - нет хранения личных данных
- Пользователь может отказаться в любой момент
- Данные используются только для исследований

## 🚀 Быстрый Старт

### Требования

- Python 3.11+
- Токен бота от [@BotFather](https://t.me/BotFather)
- (Опционально) Google Apps Script URL для логирования
- (Опционально) Docker

### Локальная Установка

```bash
# 1. Клонировать репозиторий
git clone https://github.com/yugayae/cvd-risk-bot.git
cd cvd-risk-bot

# 2. Установить зависимости
pip install -r requirements_bot.txt

# 3. Настроить окружение
cp .env.example .env
# Отредактировать .env и добавить:
# - TELEGRAM_BOT_TOKEN (обязательно)
# - GOOGLE_SHEETS_URL (опционально, для логирования)

# 4. Запустить API (терминал 1)
cd cvd-risk-api
uvicorn app.main:app --host 0.0.0.0 --port 8000

# 5. Запустить бота (терминал 2)
python bot/bot_main.py
```

### Тестирование в Telegram

1. Открыть Telegram
2. Найти вашего бота по username
3. Отправить `/start`
4. Пройти опрос

## 🌐 Развертывание

### Развертывание на Render.com (Бесплатно, 24/7)

**Рекомендуется для production**

1. **Загрузить на GitHub**
   ```bash
   git add .
   git commit -m "Initial deployment"
   git push origin main
   ```

2. **Создать аккаунт Render**
   - Перейти на [render.com](https://render.com)
   - Войти через GitHub

3. **Создать Web Service**
   - New + → Web Service
   - Подключить репозиторий
   - Runtime: **Docker**
   - Region: **Singapore** (ближе к Корее/Азии)
   - Instance Type: **Free**

4. **Настроить Переменные Окружения**
   
   | Переменная | Значение | Обязательно |
   |-----------|----------|-------------|
   | `TELEGRAM_BOT_TOKEN` | Ваш токен от BotFather | ✅ Да |
   | `GOOGLE_SHEETS_URL` | URL webhook Google Apps Script | ⚠️ Опционально |
   | `DAILY_LIMIT` | `10` (оценок в день) | ⚠️ Опционально |
   | `RATE_LIMIT_MINUTES` | `1` (пауза между запросами) | ⚠️ Опционально |

5. **Развернуть**
   - Нажать "Create Web Service"
   - Подождать 5-10 минут сборки
   - Бот будет работать 24/7! 🎉

## 📊 Производительность Модели

### Датасет

- **Источник:** Набор данных Cardiovascular Disease (Kaggle)
- **Размер:** 70,000 записей пациентов
- **Признаки:** 11 клинических параметров
- **Цель:** Бинарная классификация (CVD/Нет CVD)

### Метрики

| Метрика | Общее | Муж | Жен |
|---------|-------|-----|-----|
| **ROC-AUC** | 0.80 | 0.79 | 0.81 |
| **Чувствительность** | 75% | - | - |
| **Специфичность** | 73% | - | - |
| **Точность** | 0.72 | - | - |
| **F1-мера** | 0.74 | - | - |

### Детали Модели

- **Алгоритм:** CatBoost Gradient Boosting
- **Калибровка:** Изотоническая регрессия для надежных вероятностей
- **Интерпретируемость:** SHAP (SHapley Additive exPlanations)
- **Валидация:** Кросс-валидация со стратификацией

## 📱 Использование

### Команды Бота

- `/start` - Начать новую оценку риска
- `/help` - Показать справку и инструкции
- `/stats` - Просмотреть статистику использования
- `/cancel` - Отменить текущую оценку

### Поток Оценки

1. **Выбрать язык** (Русский / English / 한국어)
2. **Согласие на сбор данных** (опционально, анонимно)
3. **Выбрать регион** (если дали согласие)
4. **Ответить на вопросы о здоровье:**
   - Возраст (18-90 лет)
   - Пол
   - Систолическое давление
   - Диастолическое давление
   - Уровень холестерина
   - Уровень глюкозы
   - Рост (см)
   - Вес (кг) → **ИМТ рассчитывается автоматически**
   - Статус курения
   - Употребление алкоголя
   - Физическая активность

5. **Получить детальные результаты:**
   - Вероятность риска (0-100%)
   - Категория риска (Низкий / Умеренный / Высокий)
   - Топ-5 факторов риска (SHAP)
   - Обнаруженные клинические состояния
   - Уровень достоверности
   - Предупреждения безопасности
   - Медицинский дисклеймер

6. **Быстрые действия:**
   - 🔄 Новый анализ
   - 📊 Статистика
   - ❓ Помощь
