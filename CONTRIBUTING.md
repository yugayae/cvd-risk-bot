# 🤝 Contributing to CVD Risk Predictor

Thank you for contributing! 🎉

## Quick Start

1. Fork repository
2. Create branch: `git checkout -b feature/amazing`
3. Make changes
4. Test locally
5. Commit: `git commit -m "feat: add amazing feature"`
6. Push and create PR

## Code Style

- Follow PEP 8
- Use type hints
- Add docstrings
- Max line length: 100

## Testing Checklist

- [ ] Bot starts without errors
- [ ] All commands work
- [ ] Assessment flow completes
- [ ] Results display correctly
- [ ] Multi-language support intact

## Medical Code ⚠️

Changes to medical logic require:
- Clinical justification
- References to guidelines
- Extra careful review

## Questions?

Create an issue with label `question`

Thank you! ❤️
