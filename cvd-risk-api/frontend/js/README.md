# Frontend модули

Эта директория содержит модульную структуру для frontend кода.

## 🎯 Цель

Разделить большой `script.js` на логические модули для:
- Лучшей организации кода
- Упрощения тестирования
- Переиспользования компонентов
- Упрощения поддержки

## 📦 Модули

### `constants.js`
Все константы, тексты локализации, шаблоны.

### `validation.js`
Валидация данных пациента перед отправкой.

### `api.js`
Работа с API - запросы, обработка ответов.

### `utils/`
Утилиты:
- `normalization.js` - нормализация данных
- `interpretation.js` - генерация интерпретаций
- `form-handlers.js` - работа с формой
- `warnings.js` - правила предупреждений

### `render/`
Модули рендеринга:
- `patient-report.js` - отчет для пациента
- `doctor-report.js` - отчет для врача

## 🚀 Использование

См. `MODULE_STRUCTURE.md` для деталей.

## 📝 Примечание

Эти модули созданы как пример структуры. Текущий `script.js` продолжает работать. 
Модули можно использовать постепенно или полностью перейти на модульную структуру.
