# 임상 검증 및 안전성 문서  
(Clinical Validation & Safety Documentation)

> 본 문서는 임상 의사결정 지원을 목적으로 작성되었습니다.  
> 진단 또는 치료 결정을 대체하지 않습니다.  
> 법적 효력은 영어 원본 문서에 우선합니다.

---

## 1. 시스템 목적

본 시스템은 심혈관 질환(Cardiovascular Disease, CVD) 위험도를 평가하기 위한  
**임상 의사결정 지원 시스템(CDSS)** 입니다.

이 도구는:
- 임상의의 판단을 보조하기 위한 용도이며
- 진단, 치료 또는 처방을 직접 제공하지 않습니다.

---

## 2. 모델 개요

- 모델 유형: 로지스틱 회귀 (Logistic Regression)
- 입력 변수:
  - 연령
  - 수축기 / 이완기 혈압
  - 체질량지수(BMI)
  - 콜레스테롤 수치
  - 혈당 수치
  - 생활습관 (흡연, 음주, 신체 활동)
  - 성별
- 출력:
  - 심혈관 질환 위험 확률 (0–1)

---

## 3. 위험도 분류 기준

| 위험 확률 | 임상 분류 |
|---------|----------|
| < 10% | 낮은 위험 |
| 10–20% | 중등도 위험 |
| > 20% | 높은 위험 |

---

## 4. 모델 성능

모델 성능 메트릭은 독립적인 검증 데이터 세트에서 계산되었습니다:

- **ROC-AUC**: 0.800 (전체 판별 능력)
- **PR-AUC**: 0.780 (정밀도-재현율 균형)
- **민감도 (Recall)**: 95.5% (고위험 환자 식별 능력)
- **특이도**: 77.3% (저위험 환자 식별 능력)
- **정밀도 (Precision)**: 56.0% (양성 예측 가치)
- **F1-Score**: 70.6% (정밀도와 재현율의 조화 평균)
- **Brier Score**: 0.181 (보정 품질, 낮을수록 좋음)
- **성별별 ROC-AUC**:
  - 여성: 0.806
  - 남성: 0.793

이 메트릭들은 중등도-양호한 판별력과 신뢰할 수 있는 위험 보정을 나타냅니다. 모델은 고위험 사례 누락을 최소화하기 위해 민감도를 우선시합니다.

### 메트릭 기반 해석 가이드라인

- **높은 민감도 (95.5%)**는 위음성 결과를 적게 하여 스크리닝에 적합
- **중등도 특이도 (77.3%)**는 일부 위양성 결과를 초래할 수 있어 임상 판단 필요
- **Brier score < 0.2**는 우수한 확률 보정을 나타냄
- 모델 신뢰도가 낮거나 입력값이 결정 경계에 가까울 때 SHAP 설명을 신중하게 해석

---

- SHAP (SHapley Additive Explanations)
- 임상 규칙 기반 설명 (Rule-based logic)

각 예측 결과에는 다음 정보가 포함됩니다:
- 주요 위험 요인
- 위험 증가 / 감소 방향
- 임상적 해석 문구

---

## 5. 예측 신뢰도 평가 (Prediction Confidence)

예측 확률이 임상 기준점(10%, 20%)과 얼마나 떨어져 있는지를 기준으로  
신뢰도를 평가합니다.

- **높음 (High)**  
  임상 기준점과 충분히 떨어져 있어 안정적인 예측

- **중간 (Moderate)**  
  기준점과 근접 — 임상적 해석 필요

- **낮음 (Low)**  
  불확실성이 높아 추가 평가 권장

---

## 6. 안전성 검증 (Safety Layers)

### 6.1 입력 데이터 검증 (Input Sanity Check)

- 임상적으로 허용 가능한 값 범위 확인
- 데이터 타입 및 결측값 처리
- 비현실적 입력 차단

### 6.2 분포 이탈 감지 (OOD: Out-of-Distribution Check)

- 학습 데이터 분포에서 벗어난 입력 탐지
- 해당 경우:
  - 경고 메시지 출력
  - 임상의 판단 우선 권고

---

- 결과는 반드시 임상의의 판단과 함께 사용되어야 합니다
